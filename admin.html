🚀 如何更新你的網站

編輯完內容後，下載更新的 data.json 並 push 到 GitHub，網站就會自動顯示最新內容。

  1. 1 在左側選單編輯你想更新的內容區塊
  2. 2 點擊下方「下載 data.json」按鈕
  3. 3 將下載的 data.json 替換 GitHub 專案中的同名檔案
  4. 4 在終端機執行：git add . && git commit -m "update content" && git push
  5. 5 等待 1-2 分鐘，GitHub Pages 自動部署完成 ✓
